1.2. 配置

Laravel让连接和使用数据库变得异常简单.数据库配置文件是 config/production/database.php.你可以在配置文件中定义所有你的数据库连接,以及指定默认连接.这个文件中已经提供了所有支持的数据库系统连接例子.

目前Laravel支持四种数据库系统: MySQL, Postgres, SQLite, 和 SQL Server.
